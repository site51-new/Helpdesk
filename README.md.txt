¿Qué es el Sistema Web Helpdesk?

Es el Sistema Web que permite controlar las diversas incidencias técnicas ocurridas en las distintas oficinas de la 
Municipalidad Provincial de Piura(M.P.P.). Su objetivo principal es brindar un vínculo directo a los trabajadores de 
la organización con la Oficina de Infraestructura Tecnológica, permitiendo la comunicación automática y sencilla entre 
ambas entidades. Es administrada por esta última. 

En el Sistema Helpdesk encontrarás:

* Elementos digitales correspondientes al registro de incidencias y problemas técnicos suscitados dentro de la M.P.P.
* Información verificada de los usuarios registrados en el Sistema
* Páginas implementadas por la Oficina de Desarrollo de Tecnologías

Instrucciones para los usuarios:

- Para el correcto uso del Sistema, todos los usuarios deben haberse registrado en él(deben poseer usuario y contraseña).
- Para ingresar los datos en el sistema, los usuarios deben digitarlos dentro de las cajas de texto. Luego, pulsar el botón correspondiente.
- Para usar correctamente las Cajas de Combo('Combobox'), los usuarios deben hacer click sobre las cajas 
o cuadros especiales(Contienen: <<Seleccione la sede:>>,<<Seleccione la categoría:>>). A continuación, se debe 
seleccionar sólo el ítem correspondiente de entre todos los elementos.
- El Administrador del Sistema desde su sesión puede modificar algunos elementos de la Incidencia(Técnico asignado, Estado Actual).
